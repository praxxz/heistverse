# 💰 Money Heist Themed Website

A cinematic and interactive **Money Heist inspired web experience** built using **React + Vite + Framer Motion**.
This project recreates the intense and stylish vibe of *La Casa de Papel* with smooth animations, modern UI, immersive transitions, and a dark aesthetic.

---

# 🚀 Live Experience

An immersive single-page themed website featuring:

* Animated page transitions
* Character showcase
* Heist sections
* Professor section
* Smooth navigation
* Dark cinematic UI inspired by Money Heist

# ✨ Features

* 🎭 Money Heist inspired cinematic design

# ⚙️ Installation & Setup

## 1️⃣ Clone the repository

```bash
git clone https://github.com/praxxz/heistverse.git
```

## 2️⃣ Navigate into the project

```bash
cd money-heist-fixed
```

## 3️⃣ Install dependencies

```bash
npm install
```

## 4️⃣ Start development server

```bash
npm run dev
```

---

# 🏗 Build For Production

```bash
npm run build
```
